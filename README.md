# SmartCollab

SmartCollab is a Flask-based collaborative project management application.

## Features

- User authentication
- Group creation and management
- Member management
- AI contribution scoring
- Firebase integration
- Google Drive integration
- Trello integration

## Technologies Used

- Python
- Flask
- MySQL
- Firebase
- Google Drive API
- Trello API

## Installation

1. Clone the repository:

   git clone https://github.com/linuseli98-ngunjiri/smartcollab.git

2. Create and activate a virtual environment.

3. Install dependencies:

   pip install -r requirements.txt

4. Configure environment variables and service account credentials.

5. Run the application:

   python app.py
